Monogame Studio

A pocket game studio. Describe an idea — Code Buddy writes the code — play it on the spot.

Download APK · v1.0.0 Open in browser Android · ~3 MB · sideload (enable “Install unknown apps”)

What it is

Monogame Studio is the Mono game-making studio on your phone — the same editor as monogame.cc, in a native shell. Sketch a game in plain words, let the built-in AI assistant (Code Buddy) write the Lua for the Mono fantasy console (160×120, 16 shades of gray), then run it instantly. Your games sync to your account.

Mono is a tiny, constraint-driven console — small enough that a whole game fits in your head, and on your phone.

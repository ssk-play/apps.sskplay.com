AirPlay Touch

Turn your Android into an AirPlay receiver — and a touch panel for your Mac.

Get it on Google Play macOS companion install

Features

AirPlay receiver

Receive mirroring from any iPhone, iPad, or Mac on the same Wi-Fi.

Full-screen mirroring

Adaptive scaling and orientation lock for the cleanest possible picture.

PIN-protected

Optional PIN code so only trusted devices can connect.

Touch panel for Mac

Use your Android as a wireless trackpad — tap, drag, multi-finger gestures.

Gesture mapping

Two-finger swipe → scroll. Three-finger swipe → drag. Cursor snap-back.

LAN-only, no cloud

Everything runs on your local network. No accounts, no telemetry.

Install

Android

Install from the Google Play Store.

Open in Play Store

Package: com.airplaytouch

macOS companion

Required for the Touch panel feature. Install with Homebrew:

brew tap jobtools/airplay-touch
brew install --cask airplay-touch

Requirements

  • Android 7.0 or later
  • For the Touch panel: a Mac running macOS 14 or later, on the same Wi-Fi
  • DRM-protected content (Apple TV+, Netflix, etc.) cannot be mirrored — this is enforced by the source device
